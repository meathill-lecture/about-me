Bootstrap 简介

Bootstrap 是目前世界上最流行的前端框架。

由 Twitter 两名程序员创立，由开源社区共同维护。

目前经历了4个版本。

本教程将以最新的 4.0.0-alpha.6 为基础。

Bootstrap 为什么流行？

  1. 侵入性很低
  2. 文档齐全
  3. 功能丰富
  4. 活跃的社区

侵入性很低

  1. 只依赖 jQuery 和 tether
  2. 只需要引用 CSS 和 JS
  3. 几乎不影响其它库和组件

文档齐全

文档

基本涵盖了使用的方方面面

功能丰富

  1. 提共统一 UI
  2. 补强浏览器不足
  3. 直接面向移动设备，包含响应式

活跃的社区

  1. 官方范例
  2. 免费模板
  3. 收费模板
  4. 其它资源
